#Basic eCommerce Site
This is a beginner-friendly, responsive eCommerce site that I built for a Vibe Coding 101 class. It lets you browse apparel options, toss them in a cart, and pretend you're really shopping (without payment).

##Features
-Product catalog with 6 pieces of apparel
-Add and remove items from a shopping cart
-Responsive layout that works on desktop and mobile
-Simple checkout form for shipping information (just a demo)

##Technologies Used
-HTML5: Page structure and layout
-CSS3: Styling and responsive design
-JavaScript: Interactive shopping cart and form handling
-Replit: Development and hosting environment

##Live Demo
Click here to view the live site: https://tee-time-bkaston33.replit.app

-How to build responsive layouts with CSS
-How to make a basic shopping cart system
